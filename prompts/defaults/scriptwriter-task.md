# Scriptwriter Task

Write Turkish narration for the approved idea.

Requirements:

- Open with a clear hook.
- Use a Turkish title and Turkish narration even if the approved idea contains provider wording in
  another language.
- Use a cinematic, calm, accessible UykulukSciFi tone.
- Treat scientific speculation responsibly.
- Avoid unsupported certainty.
- Do not invent named people, planets, institutions, or species unless the exact name appears in the
  approved idea. Prefer stable role descriptions such as `jeolog` and keep character/setting names
  consistent across every section.
- Do not claim contamination was eliminated, an abiotic explanation was weakened, or organics imply
  biology unless the approved idea supplies verified evidence. Present these as questions or tests.
- If the approved idea contains an impossible or invented mechanism, reframe it as speculative
  fiction, a hypothesis, or a question; do not present it as established science.
- Include visual beat hints through the narration.
- End with a gentle outro or call to action.
- Target an 8-12 minute local draft.
- Produce at least 1,100 spoken narration words for local model drafts; visual directions do not
  count toward this floor. This keeps the voiceover inside the 8-12 minute target at the configured
  narration pace.
- If the output budget is tight, return a complete shorter draft with hook, body, and outro instead
  of stopping mid-sentence.
- Use Turkish production labels only: Anlatıcı: and Görsel:. Write labels as plain text without
  backticks. Never use labels such as `Narration`, `Narrator`, or `Cut to`.
- Spell production labels exactly as Anlatıcı: and Görsel:.
- Each Görsel: label controls exactly one complete sentence. Start every following sentence with
  Anlatıcı: or Görsel: so voiceover extraction cannot silently classify narration as a visual cue.
- Never invent or misspell labels such as `Anlatı:`, `Anlatyıcı:`, `Anlatici:`, `Gorsel:`,
  `Görsel -`, `Sahne:`, `Kamera:`, or `Kes:`.
- Do not repeat the same sentence, sentence skeleton, metaphor, or visual direction. If a point was
  already made, deepen it with a genuinely new beat or move to the next beat.
- Return only the final narration markdown. Do not include thinking traces or analysis blocks.
- Do not append compliance checklists, self-evaluations, word/character counts, "all requirements
  met" notes, JSON-completeness notes, or model-quality commentary.
