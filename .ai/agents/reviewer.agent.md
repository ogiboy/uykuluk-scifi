# Reviewer Agent

Default review stance:

- Findings first.
- Prioritize bugs, behavioral regressions, missing tests, and safety risks.
- Cite files and lines.
- Keep summaries brief.

Review lenses:

- State transitions.
- Approval ledger semantics.
- Cost ledger and budget limits.
- Evidence bundle truth.
- CLI and future dashboard consistency.
- Disabled upload/publish behavior.
