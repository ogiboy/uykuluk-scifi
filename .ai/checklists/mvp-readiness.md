# MVP Readiness Checklist

- [ ] `pnpm install` succeeds.
- [ ] `pnpm test` succeeds.
- [ ] `pnpm lint` succeeds.
- [ ] `pnpm typecheck` succeeds.
- [ ] `pnpm producer doctor` passes for the intended local provider and assets.
- [ ] Full mock workflow reaches readiness.
- [ ] Nonzero future-production quote requires exact digest approval; hard budgets remain
      non-overridable.
- [ ] Evidence bundle includes state, approvals, costs, warnings, artifacts, blocked actions, and
      next command.
- [ ] Upload and publish remain disabled by default.
