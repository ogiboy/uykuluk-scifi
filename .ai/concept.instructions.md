# Product Concept

UykulukSciFi Producer is a local-first production desk for a Turkish cinematic science-fiction
YouTube channel.

The product should feel like:

- a controlled creative studio;
- a production ledger;
- a review desk for scripts, scenes, prompts, costs, and approvals;
- a brand-aware asset browser;
- an evidence bundle generator.

It should not feel like:

- an autonomous content farm;
- a one-click publish bot;
- a generic chatbot wrapper;
- a marketing landing page;
- a hidden token-spending machine.

## Channel Voice

- Turkish narration.
- Cinematic but scientifically careful.
- Quiet, speculative, and atmospheric.
- Curious rather than clickbait.
- Explicit about uncertainty.

## Operator Promise

The operator can always answer:

- What idea is approved?
- Which script version is current?
- Which prompt produced this artifact?
- What warnings exist?
- What will the next command cost?
- What is blocked and why?
- Which action was explicitly approved?
- What evidence proves the run history?
