# Script Review Task

Review the generated script and produce JSON plus a Markdown summary.

Check:

- Length too short or too long.
- Unsupported scientific claims.
- Misleading certainty.
- Copyright or trademark-heavy content.
- Disaster or violent framing without responsible wording.
- Missing intro hook.
- Missing outro.
- Excessive clickbait in the title.
- Mismatch with UykulukSciFi style.
